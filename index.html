CURSO: 3RO TÉCNICO DISPOSITIVOS

EL INTERNET

INTERNET ES UNA RED GLOBAL DESCENTRALIZADA DE COMPUTADORAS INTERCONECTADAS QUE UTILIZAN PROTOCOLOS COMUNES (TCP/IP) PARA COMPARTIR INFORMACIÓN MUNDIALMENTE. SE LE CONOCE COMO LA "RED DE REDES" Y FUNCIONA CONECTANDO DISPOSITIVOS HETEROGÉNEOS —COMO COMPUTADORAS, SERVIDORES Y MÓVILES— A TRAVÉS DE CABLES, FIBRA ÓPTICA Y SATÉLITES PARA PERMITIR LA COMUNICACIÓN INSTANTÁNEA Y EL ACCESO A DATOS.

LA NUBE

LA NUBE ES UNA RED MUNDIAL DE SERVIDORES REMOTOS INTERCONECTADOS QUE ALMACENAN DATOS Y EJECUTAN APLICACIONES, PERMITIENDO ACCEDER A INFORMACIÓN DESDE CUALQUIER DISPOSITIVO CON INTERNET. EN LUGAR DE USAR EL DISCO DURO LOCAL, LA INFORMACIÓN SE GUARDA EN CENTROS DE DATOS, FACILITANDO EL TRABAJO REMOTO Y LA SINCRONIZACIÓN.

QUÉ ES UN ENLACE

UN ENLACE EN INTERNET (O LINK) ES UN ELEMENTO INTERACTIVO —TEXTO O IMAGEN— QUE CONECTA DOCUMENTOS O PÁGINAS WEB, PERMITIENDO AL USUARIO NAVEGAR HACIENDO CLIC PARA IR A OTRA UBICACIÓN. UTILIZAN HIPERVÍNCULOS (URLS) PARA CONECTAR INFORMACIÓN DENTRO DEL MISMO SITIO (INTERNO) O HACIA SITIOS EXTERNOS, FACILITANDO LA NAVEGACIÓN RÁPIDA Y ESTRUCTURANDO LA RED.

QUÉ ES UNA RED SOCIAL

LAS REDES SOCIALES SON PLATAFORMAS DIGITALES Y SITIOS WEB QUE PERMITEN A PERSONAS U ORGANIZACIONES CREAR COMUNIDADES VIRTUALES, INTERCAMBIAR INFORMACIÓN, COMPARTIR CONTENIDO MULTIMEDIA (FOTOS, VIDEOS) Y COMUNICARSE EN TIEMPO REAL. FUNCIONAN COMO ESTRUCTURAS INTERCONECTADAS DONDE LOS USUARIOS CREAN PERFILES PARA INTERACTUAR Y CONECTAR CON OTROS POR INTERESES COMUNES.

PARA QUÉ SIRVE UN ENLACE SEGURO

UN ENLACE SEGURO (COMO LOS QUE INICIAN CON HTTPS O SON VERIFICADOS POR SISTEMAS DE SEGURIDAD) SIRVE PRINCIPALMENTE PARA PROTEGER TU INFORMACIÓN PERSONAL, CONTRASEÑAS Y DATOS BANCARIOS DE CIBERDELINCUENTES. EVITA QUE LAS COMUNICACIONES SEAN INTERCEPTADAS, BLOQUEA EL ACCESO A SITIOS WEB MALICIOSOS Y PREVIENE LA DESCARGA AUTOMÁTICA DE VIRUS O MALWARE.

PARA QUÉ SIRVE UN SITIO WEB

LA FUNCIÓN PRINCIPAL DE UN SITIO WEB ES SERVIR COMO UN CENTRO DIGITAL PARA DIFUNDIR INFORMACIÓN, COMUNICAR, VENDER PRODUCTOS/SERVICIOS Y ESTABLECER PRESENCIA EN LÍNEA DE MANERA ACCESIBLE 24/7. ACTÚA COMO UNA HERRAMIENTA VERSÁTIL PARA INTERACTUAR CON USUARIOS, GENERAR CONFIANZA, AUTOMATIZAR PROCESOS COMERCIALES Y SUPERAR BARRERAS GEOGRÁFICAS.
